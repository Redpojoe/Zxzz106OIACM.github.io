# YCOI2018在线赛#17  

## 汽油补给  

[题目链接](https://www.51nod.com/onlineJudge/questionCode.html#!problemId=1288)

### 思路  

回撤贪心思想，借助单调队列（递减）。对于一个已到达的点，先将油箱填满（压入到队尾），不断使用已买到的油中最便宜的（弹出队头），用能够买到的更便宜的油替代掉队尾的节点。
想得到决策用贪心以后就比较容易了。

### 代码

```cpp
for (int i = 1; i <= n; i++) {
    scanf("%d%d", &a, &b);
    if (a > T) {
        printf("-1\n");
        return 0;
    }
    while (L <= R && Q[R].p > b) {
        v -= Q[R].v;
        R--;
    }
    Q[++R] = (node) {T - v, b};
    v = T;
    v -= a;
    while (a > 0) {
        if (Q[L].v >= a) {
            ans += a * (ll)Q[L].p;
            Q[L].v -= a;
            a = 0;
        }
        else {
            ans += Q[L].v * (ll)Q[L].p;
            a -= Q[L].v;
            L++;
        }
    }
}
```
